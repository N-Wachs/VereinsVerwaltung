# Vereinsverwaltung

Ein Desktop-Tool zur einfachen Verwaltung von Vereinsmitgliedern.
Das Projekt wurde im Rahmen eines schulischen Softwareentwicklungsprojekts erstellt.

## Überblick

Die Anwendung ermöglicht es, Mitglieder lokal zu erfassen, zu bearbeiten und zu verwalten.
Alle Daten werden ausschließlich **clientseitig verarbeitet** – es findet keinerlei Übertragung an externe Server statt.

## Datenspeicherung

* Alle Mitgliederdaten werden als **XML-Dateien** abgelegt.
* Speicherort: <Anwendungsverzeichnis>/Mitglieder
* Die Datenverarbeitung erfolgt vollständig lokal durch den Client.

## Installation & Nutzung

1. Repository klonen oder ZIP herunterladen.
2. Projekt in einer kompatiblen Entwicklungsumgebung öffnen (z. B. Visual Studio).
3. Build ausführen.
4. Die Anwendung starten – der Ordner *Mitglieder* wird bei Bedarf automatisch angelegt.
